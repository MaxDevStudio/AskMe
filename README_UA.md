# 🤖 AskMe — Інтелектуальний CLI-Диригент

**AskMe Free** — це легкий open-source AI-оркестратор для Linux. Він перетворює звичайний термінал на розумного помічника, який допомагає розробнику аналізувати код, виправляти помилки в логах та автоматизувати рутину навіть в аварійних режимах (TTY).

---

## 🚀 Основні можливості
* **Глобальний доступ:** Команда `ask` працює з будь-якої папки системи.
* **Підтримка конвеєрів (Pipes):** Передавайте вивід будь-якої команди в ШІ: `cat server.log | ask "знайди причину помилки"`.
* **Розумний пейджер:** Довгі відповіді автоматично відкриваються в `less` для зручної прокрутки.
* **Аварійний режим:** Можливість запуску прямо з GRUB для відновлення системи.
* **Розробка на ШІ:** Оптимізовано для роботи з моделлю Gemini 2.5 Flash.
