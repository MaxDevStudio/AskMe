# 🤖 AskMe — Інтелектуальний CLI-Диригент

**Репозиторій:** https://github.com/MaxDevStudio/AskMe

**AskMe Free** — це легкий open-source AI-оркестратор для Linux.  
Він перетворює термінал на розумного помічника, який може аналізувати логи, структуру проєктів, виконувати команди та допомагати в аварійних ситуаціях.

## 🚀 Основні можливості
- Глобальна команда `ask` з будь-якого місця
- Підтримка pipe: `cat log.txt | ask`, `tree | ask`, `ls -R | ask`
- Розумний пейджер (`less`) — довгі відповіді зручно прокручувати
- Аварійний режим у GRUB — завантаження прямо в TTY з AskMe
- JSON-вивід (`--json`) для зовнішніх інструментів
- Підтримка двох мов (українська + англійська)
