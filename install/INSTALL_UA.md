# 🛠 Інструкція з встановлення AskMe (Для новачків)

Цей посібник допоможе вам встановити AskMe на Linux (Ubuntu/Debian) з нуля. Навіть якщо ви ніколи не користувалися терміналом, просто йдіть крок за кроком.

---

## Крок 1: Встановлення необхідних програм
Для роботи AskMe потрібні дві речі: Node.js (двигун, який запускає наш код) та git (програма для завантаження коду).

Відкрийте термінал і вставте цю команду:
sudo apt update && sudo apt install -y nodejs npm git

---

## Крок 2: Завантаження AskMe
Тепер ми завантажимо сам проект з GitHub на ваш комп'ютер.

1. Перейдіть у папку, де ви хочете тримати проекти (наприклад, Документи):
cd ~/Documents

2. Завантажте проект:
git clone https://github.com/MaxDevStudio/AskMe.git

3. Зайдіть у папку проекту:
cd AskMe

---

## Крок 3: Встановлення внутрішніх модулів
Проекту потрібно завантажити бібліотеки для роботи з ШІ.
npm install

---

## Крок 4: Отримання ключа ШІ (API KEY)
AskMe працює через Gemini (ШІ від Google). Це безкоштовно (в межах лімітів).

1. Перейдіть на Google AI Studio: https://aistudio.google.com/app/apikey
2. Натисніть "Create API key".
3. Скопіюйте отриманий довгий код.

Тепер поверніться в термінал. Нам треба створити файл .env, де буде зберігатися цей ключ:
cp .env.example .env
nano .env

У файлі ви побачите рядок GEMINI_API_KEY=your_key_here. Видаліть your_key_here і вставте свій ключ. Натисніть Ctrl+O, потім Enter (для збереження) і Ctrl+X (для виходу).

---

## Крок 5: Створення команди ask (Глобальний виклик)
Ми зробимо так, щоб ви могли написати ask у будь-якій папці, а не тільки там, куди завантажили проект.

sudo ln -s $(pwd)/bin/ask.js /usr/local/bin/ask
sudo chmod +x $(pwd)/bin/ask.js

---

## Крок 6: Перевірка
Просто напишіть у терміналі:
ask "Привіт! Хто ти?"

Якщо ви бачите відповідь від ШІ — вітаю! Ви встановили свій власний AI-інструмент.

---

## 🗑 Як видалити
Якщо ви захочете видалити програму:
1. sudo rm /usr/local/bin/ask (видалить ярлик)
2. Видаліть папку AskMe з ваших Документів.
