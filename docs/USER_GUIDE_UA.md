# 📖 Посібник користувача AskMe

Цей документ допоможе вам зрозуміти, як ефективно використовувати AskMe у вашій щоденній роботі.

---

## 🚀 Базові команди

1. Простий запит:
ask "як оновити ядро в Ubuntu?"

2. Аналіз файлу (використання "пайпів"):
cat script.sh | ask "що робить цей скрипт?"

3. Аналіз логів:
tail -n 50 /var/log/syslog | ask "чому впав сервіс?"

---

## 🛠 Спеціальні функції

1. Розумний пейджер (less):
Якщо відповідь від ШІ занадто довга, AskMe автоматично відкриє її в режимі прокрутки. 
- Використовуйте стрілки на клавіатурі для руху вгору/вниз.
- Натисніть 'q' для виходу.
- Натисніть 's' для збереження тексту у файл (працює в TTY).

2. JSON вивід:
ask "список файлів" --json
Це корисно для програмістів, які хочуть передати відповідь ШІ в іншу програму.

3. Режим системних команд (експериментально):
Почніть запит з "cmd: ", щоб акцентувати увагу на виконанні дій.
Приклад: ask "cmd: зроби бекап папки Documents"

---

## ⌨️ Робота в аварійному режимі (TTY)
AskMe оптимізований для роботи навіть тоді, коли графічний інтерфейс Linux не завантажується.
- Alt + Shift — перемикання мови розкладки (якщо налаштовано в системі).
- Програма не потребує мишки, все керування клавішами.
