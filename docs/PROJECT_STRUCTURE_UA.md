# 📂 Структура проєкту AskMe Free

Опис організації файлів для розробників та активних користувачів.

---

## Поточна деревовидна структура (v1.5.x)

AskMe/
├── bin/
│   ├── ask.js              # Головний виконуваний файл CLI
│   └── list-models.js      # Утиліта для перевірки доступних моделей Gemini
├── docs/
│   ├── USER_GUIDE_UA.md    # Посібник користувача (UA)
│   └── USER_GUIDE_EN.md    # Посібник користувача (EN)
├── install/
│   ├── INSTALL_UA.md       # Інструкція зі встановлення (UA)
│   └── INSTALL_EN.md       # Інструкція зі встановлення (EN)
├── scripts/
│   ├── status.sh           # Допоміжний скрипт перевірки системи
│   └── hello-system.sh     # Тестовий скрипт
├── .env.example            # Шаблон для ваших ключів API
├── .gitignore              # Список файлів, які не потрапляють на GitHub
├── package.json            # Опис залежностей (Node.js)
├── README.md               # Головна сторінка проекту на GitHub
└── SPEC_FREE_UA.md         # Технічна специфікація Free версії

---

## Призначення основних папок

1. /bin — містить "мізки" програми. Саме сюди посилається команда `ask`.
2. /docs — детальна документація про те, як користуватися софтом.
3. /install — все, що потрібно для першого запуску.
4. /scripts — набір Bash-скриптів, які AskMe може пропонувати вам виконати для автоматизації.

---

## Нотатка для розробників
При додаванні нових JS-файлів у /bin, обов'язково додавайте рядок `#!/usr/bin/env node` у самий верх файлу та робіть його виконуваним через `chmod +x`.
